# Prework-study-guide

## Description

This Study Guide was created for boot camp students who were going through the Prework. it contains notes on HTML, CSS, git, and JavaScript. 
Additional purposes to the site are as follows:

- To create something useful while learning about HTML, CSS, JS, and Git.
- As a repository for notes and ideas pertaining to the UR code bootcamp.
- Provides a space where notes and ideas can be recorded and shared.

## Usage

To use this Prework Study Guide, you can review the notes in each section. For suggestions on what to study first, open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.
