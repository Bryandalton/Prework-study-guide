Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • Git workflow guide:
    1. git pull origin main Pulls latest main branch to local
    2. git checkout -b x Creates a new branch so that changes won't affect main until merge, x represents branch name
    3. Changes are made here
    4. git add -A Stages changes to local branch for commit
    5. git commit -m Commits changes locally "-m" adds message used to define changes
    6. git pull origin main Compares branch to main branch for compatablity issues
    7. git push origin x Pushes feature to remote, x represents name of branch
    8. Pull request to merge branch with main
    9. git checkout main Returns gitbash to main branch
